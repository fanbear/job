=== С ЧЕГО НАЧАТЬ ========================================================

§ Если у вас не установлен модуль OCFilter (первая установка OCFilter)

1. Установить модуль по инструкции из INSTALL

§ Если у вас установлен модуль версии 4.2 и старше

1. Перейдите в
      > Дополнения
        > Менеджер дополнений
          > Установите флажок напротив "OCFilter Modification"
          > Нажмите на "Удалить" в верхнем правом углу страницы
          > Нажмите "Очистить"
          > Затем "Обновить"

2. Установить модуль по инструкции из INSTALL

3. Установить обновление по инструкции из UPDATE

§ Если у вас установлен модуль версии 4.7.x

1. Сделать бэкап базы данных и директорий admin и catalog магазина.
2. Установить модификатор обновления из UPDATE
